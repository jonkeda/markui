# Widget Basics

Demonstrates core MarkUI widgets in markdown context.

## Buttons

```markui
[Submit]  [Cancel]  [Delete]
```

## Checkboxes and Radios

```markui
[x] Dark mode
[ ] Notifications
[-] Partial selection

---

(*) Small
( ) Medium
( ) Large
```

## Text Inputs

```markui
Username:
<________________________>

Password:
<****____________________>

Date:
<__/__/____>
```

## Links and Labels

```markui
Plain text is a label.

_Click here for details_

_Terms of Service_  _Privacy Policy_
```

## Icons and Badges

```markui
#1 Home  #2 Settings  #3 Profile  {5}  {!}
```

## Separator

```markui
First section

---

Second section
```
