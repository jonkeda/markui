# Form Patterns

Common form patterns using MarkUI.

## Simple Registration

```markui
# Create Account

Full Name:
<________________________>

Email:
<________________________>
(?) We'll send a confirmation link.

Password:
<________________________>

Confirm Password:
<________________________>

[Create Account]

_Already have an account?_
```


## Settings Form

Toggles, dropdowns, and steppers in a settings context.

```markui
+--- Preferences ---+
|
|  Theme:           <Light v>
|  Language:        <English v>
|  Font Size:       [- 14 +]
|
|  Notifications:   {[on]/off}
|  Auto-save:       {[on]/off}
|  Sound:           {on/[off]}
|
|  Volume:          [=======...] 70%
|
|  [Save]  [Reset to Defaults]
|
+--+
```
