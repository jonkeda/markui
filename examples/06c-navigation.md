# Navigation Patterns

Tabs, breadcrumbs, pagination, accordions, and tree views.

## Tab Bar

```markui
+--[[Overview]]--[Details]--[Settings]--+
|                                       |
|  Welcome to the overview tab.         |
|                                       |
|  [Learn More]                         |
|                                       |
+--+
```

## Breadcrumb

```markui
Home > Products > Electronics > Laptops
```

## Pagination

```markui
Showing 21-30 of 120 results

[<] 1  2  [[3]]  4  5  ...  12 [>]
```

## Accordion

```markui
[FAQ ^]
|  You can return items within 30 days.
+--+

[Shipping v]

[Warranty v]
```

## Tree View

```markui
- Documents
  - Work
    - Project Alpha
    - Project Beta
  + Personal
- Photos
  + 2025
  + 2026
+ Downloads
```

```markui
                   +--------------+
                   | Cut        ^X|

                   | Select All ^A|
                   +--------------+
```
